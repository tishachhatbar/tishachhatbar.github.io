# Portfolio Website Setup

## Quick Setup

1. Create repository: `tishachhatbar.github.io` (must be public)
2. Upload these files:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `profile-photo.jpg` (your photo)
3. Go to Settings → Pages → Select `main` branch → Save
4. Site live at: `https://tishachhatbar.github.io`

## Adding Your Photo

1. Rename your photo to `profile-photo.jpg`
2. Upload it with the other files
3. Photo will appear in the sidebar

## Customization

Edit `index.html` to fill in:
- About me section (line ~130)
- Contact message (line ~240)
- Your info throughout

Edit `styles.css` to change colors (lines 9-17)

## Structure

- **Sidebar:** Fixed left panel with photo, name, navigation
- **Main:** Scrolling content area on the right
- **Sections:** About, Experience, Leadership, Skills, Contact
